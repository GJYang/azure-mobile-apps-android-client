To use the Mobile Services Push Handler inside an Android app:

	- Make sure the app manifest has the required permissions to accept GCM messages
	- Create a class that extends MobileServiceNotificationHandler
	- Add the following line in the main activity:
		MobileServiceNotificationManager.handleNotifications(context, "YOUR_GCM_SENDER_ID", MyNotificationHandler.class);
	
	- Declare the following receiver and service:
	
	<receiver android:name="com.microsoft.windowsazure.mobileservices.push.MobileServiceBroadcastReceiver"
	    android:permission="com.google.android.c2dm.permission.SEND">
	    <intent-filter>
	        <action android:name="com.google.android.c2dm.intent.RECEIVE" />
	            <action android:name="com.google.android.c2dm.intent.REGISTRATION" />
	            <category android:name="[YOUR_APP_PACKAGE_NAME]" />
	    </intent-filter>
	</receiver>
	<service android:name="com.microsoft.windowsazure.mobileservices.push.MobileServiceGCMIntentService" />
